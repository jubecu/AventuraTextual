Hay dos formas de lanzar la aplicación:
- Si se quiere probar el ejemplo en bruto se debe ejecutar la clase Cliente.
- Si se quiere probar ejemplos definidos en ficheros xml se debe ejecutar la clase ReadXMLFile.

Ya hay un fichero xml para probarlo, pero se pueden definir otros siempre y cuando tengan el mismo formato.

La escena inicial debe ser la primera que se declare en el fichero, y la final la última.

Hay dos tipos de acciones: Cambiar escena y Examinar objeto.

En las acciones de tipo "Examinar objeto", en los elementos que afecta dicha acción se debe añadir
primero el propio objeto que examina.
